# HbH_WGS_GWAS

Code repository for the study:
"Characterizing clinical heterogeneity by whole genome 
sequencing of 591 patients with Hemoglobin H disease"

## Repository Structure

- 01-case-control gwas/  
  Scripts for case-control GWAS analyses

- 02-case-only gwas/  
  Scripts for case-only GWAS across 50 clinical traits

- 03-postgwas/  
  Scripts for post-GWAS analyses including fine-mapping

- 04-RVs-STAARpipeline.R  
  Rare-variant association analyses using STAAR pipeline

## Software Requirements

- PLINK2 (v2.0)
- R (v4.x)
- GCTA
- VEP (v108)
